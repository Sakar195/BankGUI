PROJECT TITLE: BankGUI
PURPOSE OF PROJECT: Done as a coursework for the second semester of Programming Module .
It is a java project done using AWT, swing libraries.
It serves as a GUI for a bank to easily create credit and debit cards
You can add credit and debit cards, set credit limit, withdraw balance from debit card and cancel credit cards.
it uses arraylist to store card details so the details are temporary
